# numguess

`<h1>numguess</h1>`

numguess는 미리 생성된 임의의 정답을 유저가 맞추는 게임입니다.

## Installation

```shell
$ pip install
```

## How to start

If you want to `emphasize` some text, just use backquotes.

```shell
$ python numguess.py
```

## Features

- Get user name
- Set answer
- Get user's guess
- Compare answer with user's guess
- Give little surprise.
